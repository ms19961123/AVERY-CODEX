# Family Flow

**Organize the chaos. Protect what matters.**

Family Flow is a production-style MVP built with Next.js App Router for overwhelmed parents coordinating school, sports, lessons, appointments, and downtime.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- lucide-react
- Framer Motion (installed, ready for tasteful interactions)
- Local mock data (no backend/auth/database)

## Installation

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Start production server

```bash
npm run start
```

## Notes

- Static-friendly architecture suitable for Vercel deployment.
- No API keys or external services required for MVP demo.
